# 人工智能导论 刷题系统

一个基于命令行的刷题工具，支持顺序练习、随机跳题、错题回顾、统计查询等功能，帮助你高效复习人工智能导论课程习题。

## 极简使用

确保你已安装 Python 3.6+，然后直接运行：

```bash
python quiz_app.py
```

程序会使用项目自带的 `questions.db` 数据库，包含已录入的题目数据。进入主菜单后根据提示选择练习模式即可。

### 主菜单说明

```
========================================
   人工智能导论 刷题系统
========================================
   题库题目数: XX
========================================
   1. 顺序练习（按题号）
   2. 随机跳题（随机选题）
   3. 只刷错题（仅显示错题）
   4. 统计信息
   5. 重置错题记录
   0. 退出
========================================
```

### 答题说明

- **单选题**: 输入选项字母（A/B/C/D）或对应数字（1/2/3/4）
- **多选题**: 用逗号、空格或分号分隔多个选项，如 `A,B,D` 或 `1,2,4`
- 大小写不敏感，输入 `q` 可随时返回菜单

## 详细内容

### 功能特性

| 功能 | 说明 |
|------|------|
| 顺序练习 | 按题号顺序逐题作答，自动从第一道未答题目开始 |
| 随机跳题 | 随机抽取题目练习，适合碎片化复习 |
| 错题练习 | 只展示曾经答错的题目，答对后自动从错题本移除 |
| 统计信息 | 查看总题数、已作答/未作答数量、总正确率、每道题正确率、Top 5 错题 |
| 重置错题 | 一键清空所有错题记录 |
| 选项打乱 | 每次展示题目时选项顺序随机，防止记忆位置 |

### 数据说明

- 数据库文件: `questions.db`（SQLite 格式）
- 数据来源: 由 `parse_and_store.py` 从 PDF 习题册解析而来
- 数据库包含三张表:
  - `questions` — 题目、选项、答案、题型（单选/多选）
  - `user_stats` — 每道题的作答次数和错题次数
  - `quiz_sessions` — 每次作答的详细记录（含时间戳）

### 从 PDF 重新解析题目

如果需要从原始 PDF 文件重新提取题目并导入数据库，请先安装依赖：

```bash
pip install -r requirements.txt
```

然后运行：

```bash
python parse_and_store.py
```

该脚本会自动读取 `人工智能导论-习题汇总-v6(1).pdf`，解析所有选择题并存入 `questions.db`。解析完成后会输出成功录入的题目数量和跳过的题目原因（记录在 `parse_log.json` 中）。

**自定义参数:**

```bash
python parse_and_store.py --pdf <PDF路径> --db <数据库路径> --log <日志路径>
```
